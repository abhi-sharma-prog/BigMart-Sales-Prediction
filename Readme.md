# 📊 BigMart Sales Analysis & Prediction Project

## 📌 Project Overview

This project focuses on analyzing BigMart sales data to identify key factors influencing product sales and to build a predictive machine learning model for sales forecasting.

The project combines:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Modeling
- Business Insights
- Interactive Dashboard (Power BI)
- Automated EDA Reports

---

## 🎯 Business Objective

- Identify factors affecting sales performance
- Analyze outlet-level performance
- Understand product category contribution
- Predict sales using Machine Learning
- Provide business recommendations

## 🧹 Data Cleaning & Preprocessing

- Handled missing values in Item Weight and Outlet Size
- Performed categorical encoding using LabelEncoder
- Applied feature scaling using StandardScaler
- Checked outliers
- Removed inconsistencies in categorical variables

---

## 📊 Exploratory Data Analysis (EDA)

- Seaborn & Matplotlib visualizations
- Correlation heatmap

Automated EDA tools used:
- Pandas Profiling

## 🤖 Machine Learning Models Used

### 1️⃣ Linear Regression
- Baseline model
- Assumes linear relationship

### 2️⃣ Random Forest Regressor
- Handles non-linear patterns
- Reduces variance using ensemble technique
- Improved prediction accuracy

---

## 📏 Model Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

Random Forest performed better than Linear Regression due to its ability to capture complex patterns.

## 📌 Key Insights

- MRP has strong positive correlation with sales.
- Medium-level priced products contribute significantly.
- Some outlet types generate higher revenue.
- Older outlets show variation in sales performance.
- Location Tier impacts overall sales.

---

## 🚀 Business Recommendations

- Focus on optimizing pricing strategy.
- Improve performance in low-performing outlet types.
- Promote high-selling product categories.
- Upgrade older outlets to improve customer experience.
